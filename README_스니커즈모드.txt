올데이픽 웹버전 스니커즈 리셀 모드 추가본

추가 기능
- 일반상품 소싱 / 스니커즈 리셀 모드 전환
- 나이키·뉴발란스·아디다스·언더아머 박스 라벨 촬영
- 브랜드·모델번호·사이즈·바코드 AI 인식
- 아울렛 가격표 촬영 후 구매가 자동입력
- KREAM 모델번호 검색 바로가기
- 즉시판매가·일반판매가·최근거래가 입력
- 수수료·배송비·기타비용 반영
- 즉시판매/일반판매 순이익과 마진율 비교
- 목표 순이익·마진율 기준 사입 추천/재검토/금지
- 사이즈 수요 위험 경고
- 동일 모델·사이즈 중복 소싱 경고
- 스니커즈 소싱기록 및 오늘 장바구니 저장

중요
- 현재 웹버전은 KREAM 가격을 자동 수집하지 않습니다.
- 모델번호로 KREAM 검색 후 해당 사이즈 가격을 입력하는 안정형 방식입니다.
- 판매 수수료율과 배송비는 실제 조건에 맞게 직접 수정하세요.

업데이트 방법
1. 압축 해제
2. GitHub 저장소에 파일 전체 덮어쓰기
3. Commit changes
4. Render에서 Manual Deploy > Deploy latest commit
5. Live 확인 후 모바일 Chrome 새로고침
