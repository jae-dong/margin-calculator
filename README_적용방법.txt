- 마지막 확인 상품의 모든 입력값과 분석결과 자동 보존

- KREAM 자동연결 유지, POIZON 수동검색 버튼 추가
- KREAM·POIZON 캡처 통합분석
- 외부 키워드 도구 버튼 및 수동 분석 버튼 제거
- 상품 인식·검색 시 키워드 경쟁/수요/검색량/판매자·상품수 자동분석


[v5.1.6 수정]
- 현재 확인 상품 초기화 시 가격표 표시가, 추가 할인율, 직접입력 할인율, 최종 매입가를 즉시 0으로 변경

- 초기화 확인창은 최초 1회만 표시하고 이후에는 바로 초기화
