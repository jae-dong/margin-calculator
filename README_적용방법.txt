올데이픽 AI 소싱계산기 WEB v2.4

추가·수정사항
- 사이즈 수요를 사용자가 선택하지 않고 거래 건수·최근 거래일·입찰가 차이로 자동 판단
- KREAM 체결거래/판매입찰/구매입찰 캡처를 최대 6장까지 한 번에 선택 및 통합 분석
- 여러 캡처에서 사이즈, 최근 거래가, 거래 날짜, 최고 구매입찰가, 최저 판매입찰가 자동입력
- 아울렛 가격표 사진에서 브랜드, 모델번호, 사이즈, 정상가, 할인가, 표시 할인율 자동인식
- 할인율을 사진 촬영 전이나 후에 바꿔도 정상가 대비 최종 매입가 즉시 재계산
- 마지막 사용한 일반상품/스니커즈 탭 유지
- KREAM 공유문구 붙여넣기 시 주소만 자동 정리
- 서비스워커 캐시 v2.4로 갱신

적용방법

Render 배포 완료 후 모바일 브라우저를 완전히 종료했다가 다시 열어 화면 상단 WEB v2.4를 확인하세요.

주의
KREAM 캡처는 같은 상품과 같은 사이즈의 화면을 함께 선택해야 정확합니다.
AI는 화면에 실제로 보이는 거래·입찰 자료를 기준으로 수요를 판단합니다.
