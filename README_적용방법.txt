올데이픽 AI 소싱계산기 WEB v5.2.5

[배포]
1. ZIP 파일을 풀고 기존 Render/GitHub 프로젝트 파일을 모두 교체합니다.
2. 재배포 후 화면 상단 WEB v5.2.5을 확인합니다.
3. 이전 화면이 계속 나오면 모바일 크롬에서 새로고침하거나 사이트 캐시를 삭제합니다.

[v5.2.5 업데이트]
- 사진 인식 상품명에서 AI 대표 검색 키워드 자동 생성
- 대표 키워드로 공개 웹 자료 자동 조사
- 월간 검색량 확인값/추정 범위, 상품수·판매자수, 수요·경쟁·소싱지수 표시
- KREAM·POIZON 캡처 분석 후 최고/평균/최저 체결가 입력칸에 즉시 반영
- 거래 목록이 없어도 화면의 최고·평균·최저 통계값을 사용
- 사이즈별 최근 거래·수익 분석 결과 자동 표시
- 캡처 분석 후 POIZON 검색 시 인식된 모델번호/상품명으로 바로 검색
- POIZON 버튼을 KREAM과 동일한 파란색으로 변경

[주의]
- 검색량·판매자 수는 공식 공개 수치가 확인되는 경우에만 확인값으로 표시합니다.
- 비공개·유료 데이터는 우회 수집하지 않으며, 공개 근거 기반 추정치는 추정으로 표시합니다.

- POIZON 버튼이 모델번호 또는 인식된 상품명으로 항상 활성화됩니다.
- POIZON 한국 사이트 검색 주소로 이동하며 검색어를 자동 전달합니다.
- 상품코드가 없으면 AI 대표 키워드·상품명·색상으로 검색합니다.
- 이동 직전 검색어를 클립보드에도 복사해 사이트가 자동검색을 막는 경우 바로 붙여넣을 수 있습니다.


[v5.2.5 캡처 분석 통합 개선]
- KREAM·POIZON 캡처를 한 장씩 고해상도(최대 2200px)로 병렬 분석합니다.
- 한 장만 성공해도 결과를 살리고, 모두 실패한 경우에만 통합 재시도합니다.
- AI는 화면 숫자 추출에 집중하고 최고·평균·최저 가격은 서버에서 직접 계산합니다.
- 사이즈가 읽히지 않아도 가격이 확인되면 분석 성공으로 처리합니다.
- KREAM 최고·평균·최저, 사이즈별 거래, 예상 순이익·ROI에 자동 반영합니다.
- POIZON 상품 자동검색 기능을 포함합니다.
- OpenAI 요청 제한시간 75초, Render Gunicorn 제한시간 120초로 조정했습니다.


[v5.2.5 사이즈별 최근 거래 수정]
- AI가 가격 배열만 반환해도 최근 거래 금액 입력칸에 자동 반영
- 날짜가 없는 거래도 금액은 보존
- 현재 입력 사이즈와 캡처 가격을 연결
- 사이즈별 최근 거래 목록을 분석 결과에 별도 표시


[v5.2.5 추가]
- KREAM 자동 이동 후 돌아와도 초기화 전까지 마지막 인식 상품 검색어와 입력값 유지
- KREAM/POIZON 버튼을 누르면 저장된 상품정보로 즉시 검색
- 초기화 버튼을 눌렀을 때만 마지막 상품 검색정보 삭제
