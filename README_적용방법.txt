올데이픽 AI 소싱계산기 WEB v5.0.9

주요 업데이트
1. 일반상품 기록 저장 버튼 제거
- 사진 분석 성공 시 자동 저장만 사용합니다.
- 촬영/선택한 상품 사진이 기록 썸네일로 함께 저장됩니다.
- 썸네일을 누르면 사진을 팝업으로 크게 볼 수 있습니다.

2. 스니커즈 키워드 경쟁·수요 분석 추가
- 일반상품과 동일한 기준으로 예상 수요도, 경쟁강도, AI 소싱지수, 예상 회전율, 추천 결과를 표시합니다.
- 브랜드, 모델번호, 색상을 우선 조합해 핵심 키워드를 자동 생성합니다.
- 사진 분석 성공 시 스니커즈 기록 자동 저장 후 키워드 분석도 자동 진행합니다.
- 분석 결과는 기록과 엑셀에 함께 저장됩니다.

3. 공통 기능 동시 반영 원칙
- 사진 자동 저장, 썸네일, 크게 보기, 키워드 분석처럼 두 상품군에 공통 적용 가능한 기능은 일반상품과 스니커즈에 함께 반영하도록 구조를 정리했습니다.

4. 기존 기능 유지
- 스니커즈 추가 할인율 0~50%, 5% 단위 선택
- 일반상품/스니커즈 사진 자동 기록
- 스니커즈 기록 상품정보 클릭 시 KREAM 검색

배포
- 기존 Render 프로젝트에 이 폴더 전체를 덮어씌운 뒤 배포합니다.
- 화면 상단 WEB v5.0.9 표시를 확인하세요.
