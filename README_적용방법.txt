올데이픽 AI 소싱계산기 WEB v3.4.1

핵심 변경
1. 일반상품 소싱과 스니커즈 리셀을 완전히 독립된 탭과 계산 흐름으로 유지합니다.
2. 일반상품 사진 또는 가격표를 카메라/사진첩에서 인식하면 상품명으로 쿠팡 모바일 검색이 자동으로 열립니다.
3. 쿠팡 자동 검색은 일반상품 탭에서만 작동하며 스니커즈 탭에는 영향을 주지 않습니다.
4. 자동 검색이 브라우저에서 차단되면 '쿠팡에서 상품 검색' 버튼으로 즉시 다시 열 수 있습니다.
5. 스니커즈의 KREAM 캡처 분석, 최고/평균/최저 체결가 계산은 기존 방식 그대로 유지합니다.
6. 마지막 사용 탭 유지 및 모바일 최적화 기능을 유지합니다.

적용
압축을 풀고 GitHub 저장소의 기존 파일을 전부 덮어쓴 뒤 Commit 합니다. Render 배포 완료 후 WEB v3.4.1 표시를 확인합니다. 기존 화면이 남으면 브라우저를 완전히 종료 후 다시 실행합니다.


수정사항: 일반상품 소싱 탭에 잘못 노출되던 KREAM 체결가 순이익 영역을 스니커즈 리셀 탭 안으로 완전히 이동했습니다.
