올데이픽 AI 소싱 WEB v2.8 적용방법

1. 이 폴더의 파일 전체를 GitHub 저장소 최상위에 덮어씁니다.
2. Commit changes를 누릅니다.
3. Render 자동 배포가 끝날 때까지 기다립니다.
4. 모바일 브라우저를 완전히 종료한 뒤 다시 실행합니다.
5. 화면 상단 WEB v2.8을 확인합니다.

주요 수정
- A상품 확인 후 B상품 사진을 불러오면 이전 상품 시세를 초기화하고 B상품 모델로 새 검색 실행
- 박스 라벨과 가격표 사진 모두 인식 완료 후 KREAM 자동연결
- Android에서는 KREAM 앱 패키지(com.fstudio.kream)를 지정해 앱 우선 실행
- 앱이 설치되지 않은 경우에만 KREAM 웹 검색으로 대체
- 같은 파일을 다시 골라도 재인식되도록 파일 선택값 초기화
