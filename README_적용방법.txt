올데이픽 AI 소싱계산기 WEB v2.6

변경사항
- 박스 라벨 또는 아울렛 가격표를 카메라로 촬영하거나 사진첩에서 선택하면 모델번호와 사이즈를 인식합니다.
- 모델번호 인식 직후 KREAM 검색 화면을 자동으로 엽니다. 주소 복사·붙여넣기는 필요 없습니다.
- 모바일 브라우저가 새 창을 차단하면 현재 화면에서 KREAM으로 이동하며, 뒤로가기로 돌아오면 인식값이 복원됩니다.
- KREAM 상품 주소 분석란은 정확한 상품 URL을 확보한 경우에만 선택적으로 사용할 수 있습니다.
- 마지막 일반소싱/스니커즈 탭 유지 기능을 포함합니다.

적용
압축을 풀고 GitHub 저장소의 기존 파일을 모두 덮어쓴 뒤 Commit 하세요. Render 배포 후 WEB v2.6 표시를 확인하세요.
