올데이픽 AI 소싱계산기 WEB v3.2

변경사항
- 사진첩 또는 카메라 인식 후 Android에서 KREAM 모바일 웹이 아닌 설치된 KREAM 앱을 우선 실행
- KREAM 공식 Android 패키지(com.fstudio.kream)를 명시한 Intent 적용
- 인식된 모델번호를 KREAM 앱 검색 주소로 전달
- KREAM 앱이 설치되지 않은 경우 Google Play 설치 화면으로 연결
- 기존 통합 사진 분석, 모델번호·사이즈·가격 인식, 마지막 사용 탭 유지 기능 포함

적용
1. 압축 해제
2. GitHub 저장소 파일 전체 덮어쓰기
3. Commit changes
4. Render 배포 완료 확인
5. 화면 상단 WEB v3.2 확인
6. 이전 화면이 남으면 브라우저 또는 설치 앱을 완전히 종료 후 다시 실행
