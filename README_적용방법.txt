올데이픽 AI 소싱계산기 WEB v2.1 최종 정리본

1. ZIP 압축을 풉니다.
2. GitHub 저장소에서 index.html.html 같은 불필요 파일을 삭제합니다.
3. 압축을 푼 폴더 안 파일 전체를 저장소 최상단에 업로드합니다.
4. Commit changes를 누릅니다.
5. Render는 자동 배포되며, 필요하면 Manual Deploy > Deploy latest commit을 누릅니다.
6. 배포 후 Ctrl+Shift+R 또는 휴대폰 시크릿 창에서 확인합니다.

기능
- 일반상품 소싱, 묶음 구성 경고, 매장별 가격기록, 재소싱 알림
- 상품 사진·가격표·영수증 AI 인식
- 기록, 실패상품, 장바구니, TOP10, 월간 통계, 자동 메모
- 스니커즈 리셀 전용 모드
- 나이키·뉴발란스·아디다스·언더아머 라벨 인식
- 모델번호·사이즈·바코드 인식
- KREAM 모델번호 검색 바로가기
- 즉시판매/일반판매 순이익 및 사입 추천 판정

중요
- KREAM 가격은 자동 수집하지 않고 직접 입력하는 안정형입니다.
- 사진 인식에는 Render의 OPENAI_API_KEY 환경변수가 필요합니다.


WEB v2.1 추가 기능
- 아울렛 정상가 입력 후 0/10/20/30/40/50% 할인 선택
- 최종 매입가와 할인액 자동 계산
- KREAM 판매자 Level 1~5 수수료 계산
- 기본 수수료 2,500원, 등급 수수료, VAT, 발송비 분리 표시
- 최근 거래 5건 날짜·금액 입력 및 평균가/추세/마지막 거래일 분석
- 가격 변동, 거래 공백, 낮은 수요, 입찰가 차이 등 위험 경고
- 실제 순이익, 정산금액, 판매 마진율, ROI 자동 계산

KREAM 실시간 가격 API는 사용하지 않습니다. 모델번호 검색 후 해당 사이즈의 가격과 최근 거래 내역을 입력하는 안정형 방식입니다.
