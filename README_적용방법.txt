올데이픽 AI 소싱계산기 WEB v3.1

주요 변경
- KREAM 공유받기 기능 제거
- KREAM 주소 복사/붙여넣기 없이 사진 여러 장 통합 분석
- 박스 라벨, 아울렛 가격표, 체결거래, 판매입찰, 구매입찰 사진 최대 10장 동시 선택
- 모델번호, 사이즈, 가격표 표시 할인가, 체결가, 최고/평균/최저 판매가 자동입력
- 기존 개별 카메라/사진첩 및 KREAM 캡처 추가분석 유지

적용
1. 압축을 풉니다.
2. GitHub 저장소의 기존 파일을 전체 덮어씁니다.
3. Commit changes를 누릅니다.
4. Render 배포 완료 후 WEB v3.1을 확인합니다.
5. 이전 화면이면 브라우저를 완전히 종료하거나 사이트 데이터를 삭제한 뒤 다시 엽니다.
