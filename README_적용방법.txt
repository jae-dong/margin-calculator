올데이픽 AI 소싱계산기 WEB v4.0.7

수정사항
- 스니커즈 라벨 또는 가격표 사진 인식 완료 후 실제 상품코드로 KREAM 검색 자동 연결
- 브라우저가 자동 연결을 차단할 경우 KREAM 수동 검색 버튼 제공
- NBPDFS 계열 내부관리번호는 검색어에서 제외

적용방법
1. 이 압축파일을 원하는 새 폴더에 압축 해제합니다.
2. GitHub Desktop에서 현재 저장소를 선택합니다.
3. 압축 해제한 파일 전체를 저장소 폴더에 덮어씁니다.
4. Summary에 WEB v4.0.7 KREAM 자동연결 수정 입력 후 Commit to main을 누릅니다.
5. Push origin을 누릅니다.
6. Render 배포가 완료될 때까지 기다립니다.
7. 휴대폰 웹앱을 완전히 종료한 뒤 다시 열고 WEB v4.0.7 표시를 확인합니다.
