ALLDAYPICK 소싱 계산기 WEB v5.4.5

주요 변경사항
- 영수증 촬영 또는 사진 불러오기 후 인식이 완료되면 즉시 자동 저장
- 영수증 기록을 일반상품·스니커즈 기록과 별도로 보관
- 기록 화면에서 영수증 매장명, 구매일, 총액, 상품내역, 인식 신뢰도 확인 및 삭제
- 영수증 기록만 별도 엑셀 다운로드 가능
- 전체 데이터 엑셀에도 '영수증 기록' 시트 포함

적용 방법
1. 기존 배포 폴더를 백업합니다.
2. 이 ZIP을 새 폴더에 압축 해제합니다.
3. 기존 파일 전체를 새 파일로 교체합니다.
4. GitHub Desktop에서 변경사항을 Commit 후 Push합니다.
5. Render 배포 완료 후 브라우저를 완전히 새로고침합니다.
6. 화면 상단 버전이 v5.4.5인지 확인합니다.


v5.4.5 추가 반영
- 네이버쇼핑 분석 탭에 별도 소싱 매장 선택 및 직접입력 추가
- 쿠팡 소싱 매장 기본값을 이마트에서 선택으로 변경
- 쿠팡·네이버쇼핑 장바구니 기록에 각 탭에서 선택한 매장이 따로 저장됨


v5.4.5 추가 반영
- 현재 확인 상품 초기화 시 일반상품 쿠팡·네이버쇼핑 판매가, 소싱가, 기타비용을 확실히 빈칸 처리
- 자동 키워드 경쟁·수요 분석의 키워드, 검색량, 판매자·상품수, 수요도, 경쟁강도, 소싱지수, 회전율을 모두 초기화
- 사진 인식 후 자동 연결 선택은 초기화 대상에서 제외
