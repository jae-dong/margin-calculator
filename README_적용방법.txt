올데이픽 AI 소싱 WEB v4.0.1.1

적용 기준
- 직장 연봉 1억원
- 쇼핑몰 연매출 5억원
- 사업 마진율 25%
- 종합소득세+지방소득세 한계세율 42%
- 부가세는 상품별 매출세액에서 공제 가능한 매입세액을 차감한 예상치

적용 방법
1. 압축을 풉니다.
2. GitHub margin-calculator 저장소의 기존 파일을 새 파일로 전부 덮어씁니다.
3. GitHub Desktop에서 변경사항을 확인합니다.
4. Summary에 WEB v4.0.1.1 세금 반영을 입력하고 Commit to main을 누릅니다.
5. Push origin을 누릅니다.
6. Render 배포 완료를 확인합니다.
7. 모바일 웹을 완전히 종료 후 다시 열고 WEB v4.0.1.1 표시를 확인합니다.

주의
상품별 부가세와 종소세는 소싱 판단용 예상치입니다. 실제 신고세액은 공제·필요경비·면세/불공제 항목 등에 따라 달라질 수 있습니다.
