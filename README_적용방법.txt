올데이픽 AI 소싱 WEB v3.0 적용방법

1. 이 폴더 안의 파일 전체를 GitHub 저장소 최상위에 덮어씁니다.
2. Commit changes를 누릅니다.
3. Render 자동 배포 완료 후 화면 상단 WEB v3.0을 확인합니다.
4. 모바일 브라우저 또는 설치 앱을 완전히 종료한 뒤 다시 실행합니다.
5. 이전 화면이 남으면 사이트 데이터/캐시를 삭제하거나 앱을 다시 설치합니다.
