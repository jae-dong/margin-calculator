올데이픽 AI 소싱계산기 WEB v3.2.1

수정 내용
- KREAM 앱 실행 실패 시 Google Play로 이동하던 오류 제거
- Android에서 KREAM 앱 연결을 우선 시도
- 앱 연결이 지원되지 않으면 KREAM 검색 웹페이지로만 이동
- KREAM 앱이 설치되어 있어도 Play 스토어로 강제 이동하지 않음
- 사진첩/카메라 인식 후 새 모델번호로 다시 검색

적용 방법
1. 압축을 풉니다.
2. GitHub 저장소 기존 파일에 전체 덮어쓰기합니다.
3. Commit changes 합니다.
4. Render 배포 완료 후 WEB v3.2.1 표시를 확인합니다.
5. 이전 화면이 남으면 브라우저 또는 설치 앱을 완전히 종료한 뒤 다시 실행합니다.
