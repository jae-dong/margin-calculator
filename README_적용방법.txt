ALLDAYPICK 소싱 계산기 WEB v5.4.0

[v5.4.0 KREAM·POIZON 수수료 개별 설정]
- 설정 화면의 스니커즈 수수료를 KREAM과 POIZON 탭으로 분리했습니다.
- 플랫폼별 판매 수수료율, 기본 수수료, 수수료 부가세율, 판매자 배송비를 각각 저장합니다.
- 플랫폼별 최소 순이익과 최소 마진율도 개별 적용합니다.
- KREAM 예상 순이익은 KREAM 설정값으로 계산합니다.
- POIZON 예상 순이익과 사이즈별 수익은 POIZON 설정값으로 계산합니다.
- 마지막으로 열었던 수수료 설정 탭을 기억합니다.
