올데이픽 AI 소싱 WEB v2.9

핵심 수정
- 안드로이드 공유 메뉴에 '올데이픽 AI소싱'이 나타나는 Web Share Target 적용
- KREAM 상품에서 공유하면 주소가 자동 입력되고 스니커즈 탭에서 분석 자동 시작
- 복사/붙여넣기 불필요

중요 설치 순서
1. 이 압축의 파일 전체를 GitHub 저장소에 덮어쓰기 후 Commit
2. Render 배포 완료 및 화면의 WEB v2.9 확인
3. 휴대폰에서 기존 '올데이픽' 홈 화면 아이콘/설치 앱 삭제
4. Chrome에서 사이트를 다시 열고 메뉴 > 앱 설치(또는 홈 화면에 추가)
5. KREAM 앱 상품 페이지 > 공유 > 더보기에서 '올데이픽 AI소싱' 선택

기존 설치본을 삭제 후 다시 설치해야 Android가 새 공유대상을 등록합니다.
일반 브라우저 바로가기만 만든 경우 공유 목록에 나타나지 않을 수 있으므로 Chrome의 '앱 설치'를 사용하세요.
