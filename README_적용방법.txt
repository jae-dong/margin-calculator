올데이픽 AI 소싱 WEB v3.3.2 모바일 최적화 수정본

수정 내용
- 모바일 상단 헤더와 일반상품/스니커즈 탭 사이 간격 확보
- 작은 화면에서 탭 글자와 버튼이 겹치지 않도록 크기 자동 조정
- 좌우 화면 넘침 방지
- 하단 고정 메뉴에 휴대폰 안전영역 여백 적용
- 입력창 확대 시 모바일 브라우저 자동 줌 방지를 위해 16px 적용
- 화면 폭 360px 이하 초소형 휴대폰 추가 대응

적용 방법
1. 압축을 풉니다.
2. GitHub 저장소의 기존 파일을 모두 이 파일들로 덮어씁니다.
3. Commit changes를 누릅니다.
4. Render 배포 완료 후 화면 상단 WEB v3.3.2을 확인합니다.
5. 기존 화면이 남으면 브라우저를 완전히 종료 후 다시 실행합니다.


[v3.3.2 변경]
- 일반상품 소싱에는 단일 쿠팡 판매가와 최종 순이익만 표시
- 최고/평균/최저 체결가 항목은 스니커즈 리셀 탭에만 표시
- 두 소싱 모드 화면이 서로 섞여 보이지 않도록 표시 분리 강화
