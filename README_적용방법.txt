올데이픽 AI 소싱계산기 WEB v5.0

주요 업데이트
1. 스니커즈 상품사진 인식 완료 후 KREAM 검색을 자동으로 별도 창에서 실행
2. 분석 준비/분석중/분석완료/분석실패 상태별 색상과 아이콘 적용
3. 분석중 모래시계 회전 애니메이션 적용
4. KREAM 여러 장 캡처 자동 압축, 1차 통합분석 실패 시 2장씩 자동 분할 재시도 후 결과 병합
5. 일반상품·스니커즈 기록에 사진 인식 원본정보와 계산 상세값 저장
6. 엑셀에 일반상품 상세, 스니커즈 상세, KREAM 사이즈별 분석, 장바구니, 세금 설정 저장
7. 전체 디자인을 밝은 블루·민트 계열로 변경
8. 쿠팡 판매가, 소싱가, 최종 매입가, KREAM 가격 입력창을 서로 다른 색상으로 구분
