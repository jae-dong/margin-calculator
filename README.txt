올데이픽 AI 소싱계산기 전체기능 업데이트

추가 기능
- 사진 한 번 촬영: 상품명 인식 → 쿠팡 검색 → 가격 후보 자동입력
- 상품명 직접 입력 검색
- 음성검색
- 바코드 촬영 검색(지원 브라우저)
- 최근 검색 100개 저장
- 즐겨찾기 100개 저장
- 홈 화면 설치(PWA)와 앱 아이콘
- 기존 실마진 계산 및 1~6개 묶음 비교 유지

업데이트 방법
1. ZIP 압축 해제
2. GitHub 저장소에서 같은 이름 파일을 모두 업로드하여 덮어쓰기
3. Commit changes
4. Render에서 Manual Deploy → Deploy latest commit
5. 배포 후 휴대폰 페이지 새로고침

Render 환경변수 유지
OPENAI_API_KEY
OPENAI_MODEL=gpt-4.1-mini
OPENAI_SEARCH_MODEL=gpt-4.1

주의
쿠팡 가격 후보는 웹 검색 기반 보조 정보입니다. 와우가, 쿠폰, 옵션, 묶음 구성에 따라 실제 가격과 다를 수 있으므로 쿠팡 화면에서 확인해야 합니다. 쿠팡 페이지는 보안 정책상 계산기 안에 직접 삽입하지 않고 새 화면으로 엽니다.
