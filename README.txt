올데이픽 AI 소싱앱 판매형 MVP

포함 기능
- 사진 AI 상품인식
- 음성 상품검색
- 텍스트 상품검색
- 바코드 촬영검색
- 쿠팡·네이버쇼핑·11번가·G마켓 검색 연결
- 쿠팡 가격 후보 자동입력
- 실마진 계산
- 1~6개 묶음 비교
- 최근검색·즐겨찾기
- 소싱기록 저장, 이익순·마진순 정렬
- 재고관리와 안전재고 경고
- 판매·실현 순이익 관리
- CSV 내보내기
- 전체 데이터 백업·복원
- PWA 홈화면 설치
- 무료체험 사용량 UI 데모

GitHub/Render 업데이트
1. ZIP 압축 해제
2. GitHub 저장소에서 파일을 업로드하여 덮어쓰기
3. Commit changes
4. Render에서 Manual Deploy > Deploy latest commit

환경변수
OPENAI_API_KEY
OPENAI_MODEL=gpt-4.1-mini
OPENAI_SEARCH_MODEL=gpt-4.1

중요한 제한
- 쿠팡은 일반 검색 결과를 앱 안에 직접 삽입하는 것을 차단할 수 있어 새 창으로 엽니다.
- 가격 자동입력은 검색 기반 후보이며 실제 옵션, 묶음 수량, 와우가, 쿠폰가를 확인해야 합니다.
- 현재 데이터는 휴대폰 브라우저에 저장됩니다. 실제 유료 판매 전에는 회원가입, 서버 DB, 결제, 약관, 개인정보처리방침, 환불·고객지원 시스템이 필요합니다.
